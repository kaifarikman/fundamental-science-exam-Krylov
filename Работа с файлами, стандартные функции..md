### **Работа с файлами**

  

**Файл** — по смыслу это способ хранить/передавать данные вне оперативной памяти программы.

  

**Классификация файлов:**

- **Внешние** — существуют до начала работы программы и/или сохраняются после её окончания. 
    
- **Внутренние** — используются как временная рабочая память, живут только во время работы программы. 
    

  

По типу компонент:

- **Текстовые** — для ввода-вывода и хранения данных, пригодных для ввода-вывода; в C для них есть предикат конца файла **EOF**. 
    
- **Нетекстовые (бинарные)** — хранят данные во **внутримашинном представлении**, экономят место и увеличивают скорость передачи. 
    

  

### **Стандартные функции**

**Режимы открытия файлов (r, w, a, r+, w+, a+)**  

**Базовый набор обычно такой:**

- открыть/закрыть: fopen, fclose
    
- чтение/запись текста: fscanf, fprintf, fgets, fputs
    
- чтение/запись бинарно: fread, fwrite
    
- позиционирование: fseek, ftell, rewind
    
- конец файла/ошибки: feof, ferror, clearerr
    
- удалить/переименовать: remove, rename
