
**Компьютерное представление целых чисел(со знаком)**

Целое число в машине представляется **машинным словом фиксированной длины** k, то есть элементом множества двоичных слов:
$W_k=\{a_1a_2\ldots a_k \mid a_i\in\{0,1\}\}.$

**Функция интерпретации** задаёт смысл слова: слово из 0 и 1 трактуется как **запись целого числа в дополнительном коде**; отдельно оговариваются «неопределённое значение» и «переполнение». Диапазон представимых значений: от -2^k-1 до 2^(k-1)-1

Также используется факт: **арифметический сдвиг** трактует слово как **дополнительный код целого числа**; при сдвиге вправо знаковый разряд остаётся неизменным.

**Компьютерное представление целых чисел(без знака)**
Беззнаковое k-битное слово обычно интерпретируют как число:
$\sum_{i=0}^{k-1} a_i\,2^i,\quad \text{диапазон } 0\ldots 2^k-1.$


**Прямой, обратный и дополнительный коды**

**1) Прямой код (sign-magnitude)**

- Старший бит — знак (0 “+”, 1 “−”), остальные биты — модуль числа.
    

  

**2) Обратный код (one’s complement)**

- Для отрицательных чисел берут положительное представление и **инвертируют все биты**.
    
- Минус: есть два нуля (+0 и −0).
    

  

**3) Дополнительный код (two’s complement)**

- Для отрицательного числа: **инвертировать биты** и **прибавить 1**.
    
- Диапазон для k бит: [-2^{k-1},\,2^{k-1}-1]. (это совпадает с тем, что закреплено в документе для целого типа)
