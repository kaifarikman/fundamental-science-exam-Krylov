# Компьютерное представление целых чисел

## Со знаком
Целое число в машине представляется **машинным словом фиксированной длины** $k$, то есть элементом множества двоичных слов:

$$W_k=\{a_1a_2\ldots a_k \mid a_i\in\{0,1\}\}.$$

**Функция интерпретации** задаёт смысл слова: слово из 0 и 1 трактуется как запись целого числа в **дополнительном коде**; отдельно оговариваются «неопределённое значение» и «переполнение». Диапазон представимых значений: $-2^{k-1} \ldots 2^{k-1}-1$.

Также используется факт: **арифметический сдвиг** трактует слово как **дополнительный код целого числа**; при сдвиге вправо знаковый разряд остаётся неизменным.

## Без знака
Беззнаковое $k$-битное слово обычно интерпретируют как число:

$$\sum_{i=0}^{k-1} a_i\,2^i,\quad \text{диапазон } 0\ldots 2^k-1.$$

## Прямой, обратный и дополнительный коды
- **Прямой код (sign-magnitude)**: старший бит — знак (0 "+", 1 "-"), остальные биты — модуль числа.
- **Обратный код (one's complement)**: для отрицательных чисел берут положительное представление и **инвертируют все биты**. Минус — два нуля (+0 и -0).
- **Дополнительный код (two's complement)**: для отрицательного числа **инвертировать биты** и **прибавить 1**. Диапазон для $k$ бит: $-2^{k-1} \ldots 2^{k-1}-1$.
